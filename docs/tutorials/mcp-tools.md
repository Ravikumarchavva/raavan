# Connect MCP Tools

MCP lets Raavan treat external tool servers as first-class capabilities.

## Basic flow

1. start or identify an MCP server
2. connect using the MCP integration layer
3. discover tools dynamically
4. hand those tool schemas to an agent

## Why use MCP

- tool discovery without hardcoding every integration
- protocol-level interoperability
- clean separation between the agent and remote tool host
